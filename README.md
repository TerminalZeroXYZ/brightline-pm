# Brightline Prediction Market

Decentralized prediction market for Brightline high-speed rail safety incidents in Florida.

Built on Polygon with USDC.

## Structure

- `frontend/` → React 19 + Vite (deploy to Vercel)
- `backend/` → Node.js + Express + Socket.io (deploy to Railway)
- `contracts/` → Solidity smart contracts (Hardhat)

## Quick Start

1. Deploy contracts to Polygon Amoy
2. Deploy backend to Railway
3. Deploy frontend to Vercel
